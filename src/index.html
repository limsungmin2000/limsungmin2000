Git setting

git init
지금 부터 이폴더에서 작업을 시작하겠다.
git add[파일명][.]
이 파일들을 버전 관리 하겠다.
git config --global user.email "limsungmin2000@gmail.com"
git config --global user.name "limsungmin"
사용자가 누구인지 기록한다.

git remote add origin[주소]

작업물을 이곳에 올릴거야[origin은 주소의 별칭이야]
git push -u origin master
로컬저장소에서 우너격저장소로 파일을 업로드 하겟다.[-u]주소를 기억해
git config --global user.email "limsungmin2000@gmail.com"
git config --global user.name "limsungmin"
사용자가 누구인지 기록한다.
